Setup Guide
Connect on Any Device
Step-by-step instructions for every platform. Get connected in minutes.
1

Download OpenVPN GUI

Go to openvpn.net/community-downloads and download the Windows Installer.
⬇ OpenVPN GUI for Windows (free)

2

Get Your .ovpn Config

Login to the portal → My Proxies → VPN → Add your proxy → Download the generated .ovpn file.

3

Import the Config

Right-click the OpenVPN GUI tray icon → Import file → Select your .ovpn file.

4

Connect

Right-click the tray icon → Select your connection → Click Connect. The icon turns green when connected.

1

Install Tunnelblick

Download Tunnelblick (free, open-source OpenVPN client for macOS).
⬇ Tunnelblick for macOS (free)

2

Download Your Config

Portal → My Proxies → VPN → Add proxy → Download .ovpn.

3

Install Config

Double-click the .ovpn file — Tunnelblick will automatically install it. Choose Only Me or All Users.

4

Connect

Click Tunnelblick in the menu bar → Select your VPN → Click Connect.

1

Install OpenVPN Connect

Download OpenVPN Connect from the App Store (free).
⬇ App Store — OpenVPN Connect

2

Download Config to iPhone

Open portal in Safari → My Proxies → VPN → tap Download .ovpn. When prompted, choose Share → OpenVPN.

3

Add Profile

In OpenVPN Connect, tap + → Import from Files → Select your .ovpn → tap Add.

4

Connect

Toggle the connection switch. Allow the VPN configuration when prompted by iOS.

1

Install OpenVPN for Android

Install OpenVPN for Android or OpenVPN Connect from Google Play (both free).
⬇ Google Play — OpenVPN for Android

2

Download Config

Open the portal in Chrome → My Proxies → VPN → Download .ovpn file. It saves to your Downloads folder.

3

Import Profile

Open OpenVPN → tap Import → browse to Downloads → select the .ovpn file → tap Import.

4

Connect

Tap the connect button. Android will ask to allow the VPN — tap OK.

1

Install OpenVPN

Run in terminal:

sudo apt install openvpn # Ubuntu/Debian
sudo yum install openvpn # CentOS/RHEL
sudo pacman -S openvpn # Arch
2

Download Config

Portal → My Proxies → VPN → Download .ovpn. Or use curl:

curl -O https://your-server/portal-app
# (download from the portal dashboard)

3

Connect

sudo openvpn --config your-config.ovpn

Or with NetworkManager GUI: Add VPN → Import from file.

4

Auto-start on Boot (optional)

sudo cp your-config.ovpn /etc/openvpn/client.conf
sudo systemctl enable openvpn-client@client
sudo systemctl start openvpn-client@client
ⓘ

Get Your L2TP Credentials First

Portal → L2TP / IPSec tab → Select a tunnel → credentials appear (Server IP, PSK, Username, Password).

W

Windows

Settings → Network → VPN → Add VPN → Provider: Windows built-in → Type: L2TP/IPsec with pre-shared key → Enter Server IP, PSK, Username, Password.



iPhone / iPad / macOS

Settings → General → VPN → Add VPN Configuration → Type: L2TP → Server, Account, Password, Secret (PSK).

A

Android

Settings → Network → VPN → + → Type: L2TP/IPSec PSK → Server, IPSec pre-shared key, Username, Password.
